This repository contains projects from CIS322 at the University of Oregon.

Contents:
README.txt - this README file.
preflight.sh - a script which will create tables in a database
sql - directory which contains files to be called with psql
src - directory which contains the files which make up the web application.
clients - directory which holds webservice clients to activate and revoke users.
clients2 - directory with modified files from clients implemented by the instructor.
util - directory which holds utility scripts.
