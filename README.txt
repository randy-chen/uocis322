This repository contains projects from CIS322 at the University of Oregon.

Contents:
README.txt - this README file.
preflight.sh - a script which will create tables in a database
sql - directory which contains files to be called with psql
src - directory which contains the files which make up the web application.
export - directory which contains the script to export data from a database and store the data in several csv files.
import - directory which contains the script to import data from csv files into a database.
clients - directory which holds webservice clients to activate and revoke users.
testdoc - directory which contains a testplan and a test report by a peer who has tested my application.
util - directory which holds utility scripts. Doesn't hold anything important as of now.
